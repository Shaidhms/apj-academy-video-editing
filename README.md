# APJ Academy Video Editing Guide

An interactive, step-by-step video editing class guide for **APJ NextGen AI** students.

**Live site:** https://shaidhms.github.io/apj-academy-video-editing/

## What's inside

A single-page interactive web guide covering:

- **VPN setup** — Proton VPN on Android, Windows, iPhone/iPad
- **CapCut** — installation in India, interface tour, splitting, keyframes, transitions, effects, picture-in-picture, text masking, walk-behind text, color grading, HSL, color recipes, audio &amp; voiceover, beat-sync editing, the Pro export trick
- **Clipchamp** — interface, splitting, transitions, color, exporting (with the keyframe limitation noted)
- **Tool comparison cheat sheet** and common mistakes

61 step-by-step screenshots and 25 hand-picked YouTube tutorials, embedded as clickable thumbnail cards.
